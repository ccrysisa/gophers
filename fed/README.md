# Fed

Fed is a Markdown editor written in Go and fyne.

## Quick start

```sh
$ go run .
```

## Install

Download fed.tar.xz from release page, then

```sh
$ tar -xvf fed.tar.xz
$ cd fed
# install to system
$ make install
# or install to local user
$ make user-install
```

If you want to uninstall

```sh
# uninstall system package
$ make uninstall
# or uninstall local package
$ make user-uninstall
```
